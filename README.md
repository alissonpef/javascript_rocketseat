## Módulos do Curso

O curso está dividido em níveis, cada um focado em uma etapa essencial do desenvolvimento:

- 🟢 **Nível 1: JavaScript Básico**
  - **Foco:** Fundamentos essenciais.
  - **Tópicos Abordados:** Fundamentos do JavaScript, Manipulação do DOM, Escopo e eventos, Depuração de código e expressões regulares.
  - **Projetos:** Conversor de moedas (`01_conversor_de_moedas`).

- 🟡 **Nível 2: JavaScript Intermediário**
  - **Foco:** Conceitos mais avançados para evoluir a partir dos fundamentos.
  - **Tópicos Abordados:** Métodos de manipulação de texto, Repetição eficiente, Manipulação de datas e horas, Programação Orientada a Objetos com classes.
  - **Projetos:** Solicitação de reembolso (`02_solicitacao_de_reembolso`).

- 🟠 **Nível 3: JavaScript antes do Framework**
  - **Foco:** JavaScript moderno e como ele prepara o terreno para o uso de frameworks.
  - **Tópicos Abordados:** Imutabilidade, módulos, e funções assíncronas, Consumo de APIs, Pacotes, compiladores e bundlers.
  - **Projetos:** Sistema de agendamento de corte de cabelo - Hair Day (`03_agendamento_hairday`).

- 🔵 **Nível 4: Conhecendo o TypeScript**
  - **Foco:** Introdução ao TypeScript abordando suas principais vantagens e motivos de uso.
  - **Tópicos Abordados:** Introdução ao TypeScript, Tipagem estática, Interfaces e tipos avançados.

## Como Iniciar

Siga as instruções abaixo para executar os projetos e códigos deste repositório localmente.

### Pré-requisitos

Certifique-se de ter o [Node.js](https://nodejs.org/) instalado, juntamente com o [NPM](https://www.npmjs.com/). Alguns projetos básicos rodam direto no navegador, então você só precisará de um navegador atualizado.

### Instalação

1. Clone o repositório
   ```sh
   git clone https://github.com/alissonpef/javascript_rocketseat.git
   ```
2. Navegue até o diretório do projeto desejado:
   ```sh
   cd javascript_rocketseat/projetos/nome_do_projeto
   ```
3. Instale as dependências (caso seja um projeto que usa NPM):
   ```sh
   npm install
   ```
4. Execute o projeto (o comando varia dependendo do projeto, ex: `npm run dev` ou abrir o arquivo `.html` no navegador).
