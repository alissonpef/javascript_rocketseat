# Bem-vindo(a) ao meu Repositório! 👋

Bem-vindo(a) à documentação do meu curso de desenvolvimento com **JavaScript** e **TypeScript**! Este repositório reflete uma jornada completa, que vai desde os fundamentos mais básicos da linguagem até a construção de aplicações complexas e o entendimento do ecossistema moderno.

Aqui, você encontrará os projetos e os códigos desenvolvidos, seguindo a progressão do curso.

---

## 📚 Módulos do Curso

O curso está dividido em níveis, cada um focado em uma etapa essencial do desenvolvimento.

### Nível 1: JavaScript Básico

Neste primeiro módulo, o foco são os **fundamentos essenciais** do JavaScript. Exploramos desde a criação de funções e a manipulação do DOM até a depuração de código, escopo, eventos e expressões regulares. O aprendizado é prático, com a construção de um conversor de moedas e um desafio opcional de lista de compras.

- **Tópicos Abordados:**
  - Fundamentos do JavaScript.
  - Manipulação do DOM.
  - Escopo e eventos.
  - Depuração de código e expressões regulares.
- **Projetos:** Conversor de moedas.
- **Tecnologias:** **JavaScript** (puro), **HTML** e **CSS**.
  
---

### Nível 2: JavaScript Intermediário

Evoluindo a partir dos fundamentos, este módulo explora conceitos mais avançados, como manipulação de textos, técnicas de repetição eficientes, datas e horas. Além disso, introduz a utilização de **classes** para organizar e estruturar o código de forma mais modular.

- **Tópicos Abordados:**
  - Métodos de manipulação de texto.
  - Repetição eficiente.
  - Manipulação de datas e horas.
  - Programação Orientada a Objetos com classes.
- **Projetos:** Sorteador de números.
- **Tecnologias:** **JavaScript** (intermediário), **HTML** e **CSS**.

---

### Nível 3: JavaScript antes do Framework

O foco deste nível é o **JavaScript moderno** e como ele prepara o terreno para o uso de frameworks. Aprenderemos funcionalidades como imutabilidade, módulos, funções assíncronas, pacotes, APIs, compiladores e bundlers, construindo uma base sólida para entender o funcionamento de frameworks como o **React** nos bastidores.

- **Tópicos Abordados:**
  - Imutabilidade, módulos, e funções assíncronas.
  - Consumo de APIs.
  - Pacotes, compiladores e bundlers.
- **Projetos:** Sistema de agendamento de corte de cabelo (Hair Day) e um desafio opcional para um Pet Shop.
- **Tecnologias:** **JavaScript** (moderno), **HTML** e **CSS**.

---

### Nível 4: Conhecendo o TypeScript

Este módulo é uma introdução ao **TypeScript**, abordando suas principais vantagens e o motivo de sua utilização. Vemos conceitos como **tipagem estática**, **interfaces** e **tipos avançados**, com o objetivo de fornecer uma base para que você consiga aplicar essa tecnologia em projetos futuros.

- **Tópicos Abordados:**
  - Introdução ao TypeScript.
  - Tipagem estática.
  - Interfaces e tipos avançados.
- **Tecnologias:** **TypeScript**.
